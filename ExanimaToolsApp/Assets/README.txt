This folder contains PNG images for Exanima equipment, including paper doll compatible armour parts for the ExanimaTools app.

- All images are 256x256 PNGs with transparent backgrounds.
- Armour images are split into paper doll parts: head, torso, arms, legs, feet, hands, etc.
- Example filenames:
    - armour_body.png
    - armour_head.png
    - armour_arms.png
    - armour_legs.png
    - armour_feet.png
    - armour_hands.png
    - shield_buckler.png
    - weapon_sword.png
    - weapon_axe.png
    - ...

You can replace these placeholders with your own art. All images are compatible with the ExanimaTools equipment system.
